# 商品上下架說明

商品都集中在 `src/products.js`。

## 上下架

每個商品都有 `status`：

- `active`：上架，客戶可以加入購物車
- `soldout`：顯示已售完，不能加入購物車
- `hidden`：下架，不會顯示在頁面

範例：

```js
{
  id: "premium-100g",
  status: "active",
  category: "特選豆",
  name: "當季特選單品",
  note: "果香、花香與明亮酸甜，批次少量烘焙。",
  image: "https://...",
  options: [
    { label: "100g", price: 160 },
    { label: "200g", price: 300 },
  ],
}
```

## 新增商品

複製一整段商品，貼到 `products` 陣列裡，然後改：

- `id`：不可重複，建議用英文與數字
- `status`：先用 `active`
- `category`：例如 `常規豆`、`特選豆`、`濾掛`
- `name`：商品名稱
- `note`：商品簡短描述
- `image`：商品圖片網址，或放在 `public/images/` 後用 `${imageBase}images/檔名.jpg`
- `options`：規格與價格

## 濾掛或不需要研磨的商品

加上：

```js
noGrind: true
```

頁面就會自動顯示「不需研磨」。
