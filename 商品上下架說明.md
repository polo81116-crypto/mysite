# 商品上下架說明

現在商品資料集中在 `src/products.json`，可以用 `商品管理工具.html` 編輯。

## 使用商品管理工具

1. 雙擊開啟 `商品管理工具.html`。
2. 按「開啟商品資料檔」。
3. 選擇 `src/products.json`。
4. 左邊選商品，右邊修改品名、分類、價格、規格、圖片與狀態。
5. 按「套用修改」。
6. 按「儲存」。
7. 儲存後再把網站更新到 GitHub。

建議用新版 Chrome 或 Edge 開啟，才可以直接儲存回原本的 JSON 檔。

## 上下架狀態

- `active`：上架，客戶可以加入購物車
- `soldout`：顯示售完，不能加入購物車
- `hidden`：下架，不會顯示在頁面

## 圖片填寫方式

如果圖片放在 `public/images/`，填：

```text
images/檔名.jpg
```

如果是網路圖片，填完整網址：

```text
https://example.com/image.jpg
```

## 不需要研磨的商品

濾掛或即飲類商品，把「不需要研磨」選成「是」即可。
