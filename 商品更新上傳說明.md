# 商品更新上傳說明

平常更新商品的流程：

1. 開啟 `商品管理工具.html`。
2. 開啟 `src/admin-products.json`。
3. 修改商品、價格、圖片、上下架狀態。
4. 儲存回 JSON。
5. 雙擊 `一鍵上傳商品更新.bat`。
6. 輸入這次更新說明，例如 `上架新豆單`。
7. 等 GitHub Pages 自動部署完成。

如果只是想先存本機，不要雙擊上傳腳本即可。

## 圖片

商品圖片可以填：

```text
Imgea/ChatGPT Image 1.png
```

或：

```text
images/medium-dark-1lb.jpg
```

若新增圖片到 `Imgea` 資料夾，商品管理工具裡可以按「選擇 Imgea 資料夾」重新讀取圖片清單。

## 食品標示欄位

商品管理工具已新增：

- `產地`：先填實際咖啡豆產地；若還沒整理完成，可暫填「依商品標示為準」。
- `保存方式`：預設為「請置於陰涼乾燥處，開封後請密封保存並盡早飲用。」
- `負責廠商資訊`：目前預設為「豆之楓企業社」。

這三欄會顯示在商品列表，也會一起存入 `src/admin-products.json`。

## 特價商品

商品管理工具裡的「商品折扣％」是單品折扣，例如填 `20` 代表該商品先打 8 折。

如果該商品已經是特價品，可以再勾選：

- `特價商品不再套全館 9 折`：加入購物車後，不會再被全館折扣重複折價。
- `不列入滿千免運門檻`：該商品金額不會計入滿 1,000 元免運。
